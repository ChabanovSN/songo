## Requirements
[Java Runtime Environment](http://java.com)
### Windows
None
### Linux
* gtk2
* libwebkitgtk
* libmpg123

### Mac OS
* [mpg123](http://braumeister.org/formula/mpg123) can be installed from [homebrew](http://mxcl.github.com/homebrew/)
